# Product Manager Portfolio

A professional portfolio website showcasing my experience as a Product Manager, built with Next.js and Tailwind CSS.

## Features

- 📱 Responsive design optimized for all devices
- 🎯 Showcase for product management case studies
- 📊 Project highlights and metrics
- 📝 Blog section for sharing insights
- 🔍 SEO optimized
- ⚡ Fast performance with Next.js

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework for production
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org/) - Static type checking
- [ESLint](https://eslint.org/) - Code linting
- [Prettier](https://prettier.io/) - Code formatting

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/yourusername/pm-portfolio.git
cd pm-portfolio
```

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
pm-portfolio/
├── src/
│   ├── app/           # Next.js app directory
│   │   ├── about/     # About page
│   │   ├── projects/  # Projects showcase
│   │   ├── blog/      # Blog posts
│   │   └── page.tsx   # Homepage
│   └── types/         # TypeScript type definitions
├── public/            # Static assets
└── tailwind.config.js # Tailwind CSS configuration
```

## Customization

1. Update the content in `src/app/page.tsx` with your personal information
2. Modify the navigation links in `src/app/layout.tsx`
3. Add your case studies in the `src/app/projects` directory
4. Write blog posts in the `src/app/blog` directory
